Faucet Description

The Zap Testnet Faucet is a developer tool able to send 1K ERC20 and 500K BEP20 test ZAP to users MetaMask accounts. Copy the Zap Token Contract address that corresponds to the network your testing above and add it to your MetaMask account and start the transaction by clicking the Get ZAP button.
