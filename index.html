Faucet Description

The Zap Testnet Faucet is a developer tool able to send 1K ERC20 and 500K BEP20 test ZAP to a users MetaMask account. Copy the Zap Token Contract address that corresponds to the network your testing and add it to your MetaMask account and receive tokens by clicking the Get ZAP button.
